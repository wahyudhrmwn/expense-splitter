# Expense Splitter App

Aplikasi untuk membagi tagihan dengan teman secara adil dan otomatis.

## Fitur

- Buat group untuk membagi pengeluaran
- Tambah anggota ke dalam group
- Tambah pengeluaran dengan detail per item per orang
- Support pajak (PPN) dengan persentase custom
- Hitung saldo setiap orang secara otomatis
- Tampilkan kesimpulan penyelesaian hutang piutang

## Teknologi

- Next.js 16
- React 19
- TypeScript
- Zustand (State Management)
- Tailwind CSS
- Lucide Icons

## Instalasi

```bash
npm install
```

## Menjalankan Development Server

```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Build untuk Production

```bash
npm run build
npm start
```
